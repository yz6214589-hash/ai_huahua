# sentiment-analysis 技能指南

## 适用场景

### 微观舆情（个股/行业）
- 监控某只股票或行业的近期新闻舆情
- 分析市场整体情绪（贪婪/恐慌）
- 捕捉"资产重组"、"业绩预增"等关键事件
- 生成事件驱动的交易信号和机会提示

### 宏观风险（全球/地缘政治）
- 查看VIX恐慌指数、美国10年期国债收益率等宏观指标
- 监控Polymarket预测市场的地缘政治事件（战争、停火、制裁、关税等）
- 获取综合恐慌/贪婪评分，判断当前是否适合入场
- 风险传导分析：OVX与VIX是否同步共振

## 数据来源

### 微观数据
- 东方财富个股新闻（通过 akshare）
- 上市公司公告
- 央视新闻（政策面）

### 宏观数据
- VIX / OVX / GVZ 波动率指数（akshare / yfinance）
- 美国10年期国债收益率（akshare / yfinance）
- Polymarket 预测市场（Gamma API，免费无需认证）
- A股情绪指标：上证指数、北向资金

## 依赖技能
- **read-pdf**: 事件触发后可联动 read-pdf 进行深度财报分析
- **write-report**: 可联动 write-report 生成完整的事件驱动分析报告
- **web-search**: 可联动 web-search 搜索事件的更多背景信息

## 可用脚本

### 微观舆情脚本

| 脚本 | 功能 | 参数 |
|------|------|------|
| `scripts/news_fetcher.py` | 抓取新闻并按关键词过滤 | `--stock`(股票代码), `--keywords`, `--days` |
| `scripts/sentiment_scorer.py` | LLM 情感分析与评分 | `--news_file`, `--output_dir` |
| `scripts/event_detector.py` | 事件识别与交易信号 | `--news_file`, `--output_dir` |

### 宏观风险脚本

| 脚本 | 功能 | 参数 |
|------|------|------|
| `scripts/market_fear_index.py` | 恐慌/贪婪指数监控（VIX+国债+综合评分） | `--output_dir`, `--include_ashare` |
| `scripts/polymarket_monitor.py` | Polymarket预测市场监控 | `--keyword`, `--keywords`, `--min_volume`, `--output_dir` |

## 关键词过滤体系

### 利好类
资产重组、回购、业绩预增、股权激励、大额订单、战略合作、增持

### 利空类
业绩预减、违规处罚、股东减持、商誉减值、退市风险、诉讼仲裁

### 政策类
降准降息、产业政策、监管新规、财政刺激、贸易政策

## 宏观指标阈值参考

### VIX 恐慌指数
| 区间 | 含义 | 操作建议 |
|------|------|----------|
| < 15 | 极度平静（警惕自满） | 市场可能过于乐观 |
| 15-20 | 正常 | 按策略正常操作 |
| 20-25 | 焦虑 | 适当控制仓位 |
| 25-35 | 恐慌 | 关注超跌反弹机会 |
| > 35 | 极度恐慌 | 历史上往往是中长期买入良机 |

### 美国10年期国债收益率
| 区间 | 含义 | 资产配置 |
|------|------|----------|
| > 4.4% | 利率偏高 | 看好价值股和防御板块 |
| 4.3-4.4% | 分水岭 | 密切关注方向 |
| < 4.3% | 宽松预期 | 资金回流成长股 |

### 风险传导逻辑（开源证券研报）
- OVX飙升但VIX滞后 -> 风险集中在能源端，尚未传导至全球信用风险
- OVX与VIX同步共振向上 -> 地缘风险已触发流动性危机，需立即风控

## 工作流程

### 微观舆情流程
1. 使用 `news_fetcher.py` 获取目标股票/行业的近期新闻
2. 使用 `sentiment_scorer.py` 对新闻进行情感分析
   - 每条新闻输出：情感倾向（正面/负面/中性）、强度（1-5）、关键实体
   - 汇总生成情绪指数
3. 使用 `event_detector.py` 识别重大事件并生成交易信号
   - 事件分类：利好/利空/政策
   - 交易信号：买入/卖出/观望
4. 如发现重大事件，可联动其他技能进行深度分析

### 宏观风险流程
1. 使用 `market_fear_index.py` 获取全球恐慌指标
   - VIX / OVX / GVZ 波动率
   - 美国10年期国债收益率
   - 综合恐慌/贪婪评分（0-100）
2. 使用 `polymarket_monitor.py` 监控预测市场
   - 地缘政治事件概率（战争/停火/制裁等）
   - 聪明钱信号检测
   - 资产配置建议
3. 结合微观舆情和宏观风险，给出完整的市场研判

## 示例对话

用户: "帮我看看比亚迪最近有什么新闻"
步骤:
1. 执行 `news_fetcher.py --stock 002594 --days 7`
2. 执行 `sentiment_scorer.py --news_file data/002594_news.json`
3. 返回新闻摘要和情绪分析结果

用户: "最近A股有没有资产重组的消息"
步骤:
1. 执行 `news_fetcher.py --keywords 资产重组 --days 3`
2. 执行 `event_detector.py --news_file data/资产重组_news.json`
3. 返回事件列表和交易机会提示

用户: "现在市场恐慌指数多少？适合入场吗？"
步骤:
1. 执行 `market_fear_index.py --include_ashare`
2. 返回VIX数值、综合恐慌/贪婪评分、操作建议

用户: "看看Polymarket上关于中美关税的预测"
步骤:
1. 执行 `polymarket_monitor.py --keyword "tariff" --min_volume 100000`
2. 返回相关预测市场概率、交易量、资产配置建议

用户: "发现比亚迪有资产重组，帮我做个深度分析"
步骤:
1. 联动 read-pdf 技能读取比亚迪最新财报
2. 联动 write-report 技能生成五步法分析报告
3. 将舆情分析与基本面分析结合，给出完整建议
