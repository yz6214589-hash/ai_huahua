

# 5. 舆情获取和金融情感分析

## 5.1 金融舆情基础认知

金融舆情是市场参与者对个股、行业及整体市场的集体情绪与看法，是驱动股价短期波动的核心因素之一。华尔街经典投资逻辑**Buy the rumor, sell the news（买在传言时，卖在消息落地时）**，体现舆情对股价具备前瞻性影响，多数情况下股价在正式公告发布前已完成预期反应。

机构与散户存在明显信息差：机构配备24小时舆情监控系统，可实时扫描新闻、公告、社交媒体，第一时间捕捉资产重组等关键事件；散户通常在股吧讨论发酵、股价涨停后才获知消息，信息获取严重滞后。通过AI实现新闻自动抓取、情感自动分析、交易信号自动生成，能够有效缩小信息差。

## 5.2 舆情数据获取

### 5.2.1 舆情数据三层结构

| 层次   | 数据源            | 时效性       | 可靠性      |
| ---- | -------------- | --------- | -------- |
| 一手信息 | 上市公司公告、交易所公告   | 最快（盘后/盘中） | 最高（法定披露） |
| 二手信息 | 东方财富、证券时报等财经新闻 | 快（分钟级）    | 较高（记者采编） |
| 三手信息 | 股吧、社交媒体、自媒体    | 最快（秒级）    | 较低（噪音多）  |

实战中优先采用**一手、二手数据**，通过akshare金融数据工具库获取东方财富新闻与上市公司公告。

### 5.2.2 核心数据接口

akshare是免费Python金融数据工具库，核心接口如下：

* `stock_news_em(symbol)`：获取东方财富个股新闻，示例：`stock_news_em("002594")`

* `stock_notice_report(symbol)`：获取上市公司公告，示例：`stock_notice_report("002594")`

* `news_cctv(date)`：获取央视政策新闻，示例：`news_cctv("20260330")`

### 5.2.3 新闻筛选与处理

#### 关键词过滤体系

* 利好类：资产重组、回购、业绩预增、股权激励、大额订单、战略合作、增持、扭亏为盈

* 利空类：业绩预减、违规处罚、股东减持、商誉减值、退市风险、诉讼仲裁、亏损

* 政策类：降准降息、产业政策、监管新规、财政刺激、贸易政策

#### 时间范围筛选

默认保留最近7天新闻，支持自定义时间周期，过滤过期无效信息。

#### 新闻去重

按新闻标题去重，避免同一事件被重复分析，提升数据处理效率。

## 5.3 金融情感分析

### 5.3.1 学术与机构研究支撑

1. 哥伦比亚大学Tetlock教授（2007）研究证实：媒体悲观情绪指数可预测次日股市下跌与交易量上升，极端乐观/悲观情绪对应市场高活跃度，该研究获金融学顶级Smith-Breeden奖。

2. 摩根大通Volfefe Index（推特波动率指数）：追踪特朗普推文对国债波动率的影响，命中率超60%；含宏观经济、贸易内容的推文可引发美元升值、外汇波动率下降，基于此构建的策略夏普比率达0.69。

### 5.3.2 LLM情感分析流程

#### Prompt设计

以专业金融舆情分析师视角，对新闻进行结构化分析，输出包含情感倾向、强度、相关实体、核心关键词、摘要、市场影响的JSON格式结果。

#### 执行步骤

1. 抓取个股指定周期新闻，生成JSON数据文件

2. 调用大模型对新闻逐条进行情感评分

3) 聚合所有分析结果，生成恐惧贪婪指数（Fear & Greed Index）

### 5.3.3 恐惧贪婪指数

参考CNN恐惧贪婪指数，分值0-100：

* 0-24：极度恐惧

* 25-49：恐惧

* 50：中性

* 51-74：贪婪

* 75-100：极度贪婪

指数通过正面新闻占比映射计算，结合大模型总结市场核心主题、风险点与交易机会。

### 5.3.4 应用场景

1. 逆向交易策略：极度恐惧（≤25）为买入信号，极度贪婪（≥75）为卖出/减仓信号

2. 量化交易回测：指数≤25时开期货多头，策略在在多国股指期货上测试显示盈利因子为正

3) 买卖点确认：结合支撑位、阻力位、RSI、MACD等技术指标优化入场出场时机

   * 买入确认：指数 ≤25 + 价格触及强支撑位 + RSI <30 + MACD金叉

   * 卖出确认：指数 ≥75 + 价格触及阻力位 + RSI >70 + MACD死叉

4) 风险管理：极端情绪区降低仓位，中性区减少交易，依据波段高低点设置止损

   * 极端情绪区：降低仓位规模（高波动性）&#x20;

   * 中性区（40-60）：避免交易（信号不明确）&#x20;

   * 止损设置：基于近期波段高低点或ATR设置止损

# 6. 事件驱动和恐慌指数分析

## 6.1 事件驱动交易系统

### 6.1.1 事件驱动核心逻辑

事件驱动交易是对冲基金与量化机构常用策略，核心是识别与股价变动显著相关的特定事件，在市场充分反应前建仓获取超额收益。与趋势跟踪互补：趋势跟踪关注价格方向（技术面），事件驱动关注影响价格的核心事件（消息面）。

### 6.1.2 A股典型事件与市场影响

| 事件类型 | 历史影响            | 交易策略            |
| ---- | --------------- | --------------- |
| 资产重组 | 复牌首日平均涨幅8%-15%  | 事件确认后关注，警惕利好出尽  |
| 回购增持 | 中期正面效应（3-6个月）   | 跟随大股东，重点关注实控人增持 |
| 业绩预增 | 短期正面脉冲（1-5个交易日） | 预增幅度＞50%更具价值    |
| 股东减持 | 短期负面压力          | 回避或减仓           |
| 违规处罚 | 可能连续跌停          | 立即回避            |

### 6.1.3 事件识别与信号生成

#### 两层检测策略

1. 关键词快速检测：毫秒级过滤，基于预定义关键词库批量筛选新闻

2. LLM精细识别：秒级分析，理解复杂语义与隐含事件，弥补关键词检测不足

#### 交易信号映射

利好事件对应强烈关注、看多信号；利空事件对应谨慎、看空、强烈回避信号，结合事件类型给出明确操作依据。

## 6.2 宏观恐慌指数监控

### 6.2.1 核心恐慌指数解析

#### VIX（CBOE波动率指数）

* 本质：标普500指数期权隐含波动率，代表市场对冲风险的“保险价格”

* 历史极值：金融危机89.53、疫情82.69，正常牛市12-18

* 相关性：与标普500呈-0.8负相关，极端恐慌后常出现均值回归买入机会

#### OVX与GVZ

* OVX（原油波动率指数）：原油ETF期权隐含波动率，飙升代表地缘政治紧张、供应链危机

* GVZ（黄金波动率指数）：黄金ETF期权隐含波动率，升高反映市场避险情绪升温

#### 美国10年期国债收益率

全球资产定价之锚，无风险利率基准；收益率上升压制高估值成长股，利好价值股，4.3%为关键分水岭。

### 6.2.2 多维度恐慌指标体系

#### VIX阈值与操作建议

* ＜15：极度平静，警惕市场过度乐观

* 15-20：市场正常，按策略执行

* 20-25：市场焦虑，适当控制仓位

* 25-35：市场恐慌，关注超跌反弹

* ＞35：极度恐慌，中长期买入良机

#### 风险传导逻辑

* OVX飙升、VIX滞后：风险集中在能源端，未传导至全球宏观

* OVX与VIX同步共振：地缘风险引发流动性危机或衰退预期，立即执行风控

#### 综合评分机制

整合VIX、美国10年期国债收益率、OVX、GVZ等指标，生成0-100分综合恐慌/贪婪指数，直观判断市场整体情绪。

## 6.3 Polymarket预测市场

### 6.3.1 核心原理

基于有效市场假说，大量参与者用真金白银押注事件结果，市场价格自动汇聚为最准确概率估计，预测准确度远超传统民调与专家判断。Polymarket是全球最大去中心化预测市场，基于Polygon区块链运行，支持实时链上查询。

### 6.3.2 市场应用价值

1. 前瞻性判断：弥补传统新闻回顾性分析短板，提前捕捉未来事件概率变化

2. 聪明钱信号：单笔大额押注、交易量超均值3倍、概率＞85%，大概率伴随内幕信息流通

3) 事件概率与资产配置

   * 开战概率＞60%：做多黄金、原油，做空科技股

   * 停火概率快速上升：平仓原油多头，布局风险修复资产

   * 概率长期僵持（40%-60%）：做多波动率

   * 关税概率＞60%：回避出口企业，关注内需板块
