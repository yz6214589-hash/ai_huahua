## 功能模块

### 1. 主力活动列表

**功能：**
- 展示所有主力活动记录
- 支持按股票代码、名称、活动类型筛选
- 支持按日期范围筛选
- 显示字段：日期、股票代码、名称、类型、成交量、成交金额、大单占比、主力类型

**主力类型：**
- `institution` - 机构主力
- `hot_money` - 游资
- `retail` - 散户

### 2. K线图标注

**功能：**
- 显示股票K线图
- 绿色向上箭头(▲)表示主力买入
- 红色向下箭头(▼)表示主力卖出
- 鼠标悬停查看详细信息

**使用ECharts实现，支持：**
- 缩放
- 数据区域选择
- 交互式提示

### 3. 风控规则配置

**默认规则：**

1. **成交量异常告警**
   - 阈值：2.0倍
   - 说明：当日成交量超过过去5日平均成交量的2倍
   - 状态：已启用

2. **大单卖出告警**
   - 阈值：500,000元
   - 说明：单笔大单卖出超过50万元
   - 状态：已启用

3. **主力资金净流出告警**
   - 阈值：100,000,000元
   - 说明：主力资金净流出超过1亿元
   - 状态：已启用

4. **持仓比例异常告警**
   - 阈值：15%
   - 说明：主力持仓比例变化超过15%
   - 状态：已禁用

**操作：**
- 启用/禁用规则
- 修改阈值
- 自动保存到数据库

### 4. 统计数据展示

**统计卡片：**
- 今日买入次数
- 今日卖出次数
- 净流入金额
- 活跃规则数量

**功能：**

## API接口

### 主力活动API

- `GET /api/mainforce/activities` - 获取活动列表
  - 参数：stock_code, activity_type, mainforce_type, start_date, end_date, alert_status, page, page_size
  - 返回：活动列表数组

- `POST /api/mainforce/activities` - 创建活动
  - 请求体：MainForceActivity对象
  - 返回：{id, message}

- `GET /api/mainforce/activities/{id}` - 获取单个活动
  - 返回：MainForceActivity对象

### 告警规则API

- `GET /api/mainforce/rules` - 获取规则列表
  - 参数：enabled
  - 返回：规则列表数组

- `PUT /api/mainforce/rules/{id}` - 更新规则
  - 请求体：MainForceRule对象
  - 返回：{message}

- `POST /api/mainforce/rules/{id}/trigger` - 触发规则检查
  - 参数：stock_code, stock_name, value
  - 返回：{triggered, rule_name, threshold, actual_value, message}

### K线标注API

- `GET /api/mainforce/markers` - 获取标注列表
  - 参数：stock_code, marker_type, start_date, end_date
  - 返回：标注列表数组

- `POST /api/mainforce/markers` - 创建标注
  - 请求体：KlineMarker对象
  - 返回：{id, message}

### 统计API

- `GET /api/mainforce/summary` - 获取统计摘要
  - 返回：MainForceSummary对象

- `GET /api/mainforce/statistics` - 获取统计列表
  - 参数：start_date, end_date
  - 返回：统计列表数组

## 故障排除

### 问题1：后端无法启动

**错误：** `ModuleNotFoundError: No module named 'api.mainforce'`

**解决：** 检查 `backend/app.py` 中是否正确导入了mainforce路由

### 问题2：前端无法连接后端

**错误：** `Failed to fetch` 或网络错误

**解决：** 
1. 确认后端服务正在运行（http://localhost:8000）
2. 检查CORS配置
3. 检查防火墙设置

### 问题3：数据库连接失败

**错误：** `sqlite3.OperationalError` 或连接超时

**解决：**
1. 检查数据库文件是否存在：`ls -la /Users/apple/Desktop/ai_huahua/ai_quant/backend/risk_management.db`
2. 检查文件权限
3. 执行迁移脚本

### 问题4：页面显示默认数据而不是真实数据

**原因：** API不可用或返回错误

**解决：**
1. 检查后端服务是否正常运行
2. 查看浏览器控制台错误信息
3. 检查数据库迁移是否成功执行
