### 3.2 边界场景清单

撰写过程中需覆盖以下边界场景：

**数据模块：**
- 数据集查询结果为空时的展示
- 导出数据超过10000行时的分批提示
- 股票搜索防抖与超时处理（150ms debounce / 1.2s timeout）

**研报模块：**
- LLM 调用超时（默认300s任务超时 / 90s LLM超时）
- FAISS 索引不存在时降级为静态配置
- 同一股票重复提交研报任务的处理
- Worker 线程异常退出的任务恢复

**交易模块：**
- QMT 网关连接超时（默认60s可调）
- 风控连续否决达到重试上限（max_retry=2）的流程终止
- 模拟账户（sim_account）与真实账户的隔离

**通用场景：**
- 网络中断/弱网环境下的前端错误处理
- API Key 认证失败时的响应
- 并发请求下的速率限制（10s/200次）
