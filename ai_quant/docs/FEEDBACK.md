## 字段说明

| 字段 | 说明 |
|------|------|
| **反馈编号** | 唯一标识，格式：FB-模块-序号，如：FB-RP-001 |
| **模块** | 见下方模块列表 |
| **问题类型** | 功能缺陷/体验优化/性能问题/安全漏洞/数据问题/其他 |
| **严重程度** | Critical/Major/Minor/Enhancement |
| **优先级** | P0/P1/P2/P3 |
| **状态** | Open/In Progress/Resolved/Closed/WontFix |

**模块代码：**
- HOME=首页总览
- RP=智能研报
- DATA=数据与交付
- JOB=采集任务
- SENT=舆情监控
- MORN=晨会简报
- RISK=风控中心
- EXEC=执行监控
- WATCH=自选股
- STRAT=策略分析
- CHAT=AI对话
- API=接口
- DB=数据库
- PERF=性能
- SEC=安全
- DOC=文档
